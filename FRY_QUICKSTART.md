## 1) Install + Configure

**Step 1 — Install**
```bash
gemini extensions install https://github.com/galz10/bender-extension
```

**Step 2 — Configure**
Update `~/.gemini/settings.json` to enable hooks, skills, and safe git usage:
```json
{
  "hooks": { "enabled": true },
  "experimental": { "skills": true },
  "context": {
    "includeDirectories": ["~/.gemini/extensions/bender"]
  },
  "tools": {
    "exclude": ["run_shell_command(git push)"],
    "allowed": [
      "run_shell_command(git commit)",
      "run_shell_command(git add)",
      "run_shell_command(git diff)",
      "run_shell_command(git status)"
    ]
  }
}
```

**Step 3 — Run safely**
Use a sandbox. Enable **YOLO mode** (`-y`) to reduce prompts:
```bash
gemini -s -y
```

## 2) Pick a Mode

| Command | Best for |
| :--- | :--- |
| **`/bender "task"`** | A clear, direct task with known outcomes. |
| **`/bender-prd "idea"`** | Complex ideas that need a PRD first. |

---

## 3) Examples

### Quick Fix
**Prompt:**
```bash
/bender "Read this github issue [PASTE CONTENT] and apply the fix. Validate by running npm run build and npm test."
```
**Why it works:** The goal and verification steps are explicit.

### New Feature
**Prompt:**
```bash
/bender-prd "I want to add ZSH-style tab completion to the CLI. If I type 'ls' and tab, show files. Validate by running npm run build and npm test."
```
**Why it works:** Ambiguous UX needs a guided PRD interview.
